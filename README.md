# Green Trees Initiative Website

## Project Summary

This is a multi-page website for the Green Trees Initiative, a climate-action organization focused on combating climate change and creating sustainable livelihood opportunities for women in developing countries.

The site includes two main pages:
- **Home Page** (`index.html`): Presents the organizationâ€™s mission, highlights key initiatives, features impact stories and news, and includes clear calls to action for involvement and donations.
- **Donation Page** (`donation.html`): Provides a secure donation form along with information on how donations are used and assurances about payment and data security.

## Key Features

- Mobile-first responsive design built with Flexbox and CSS Grid
- Accessibility improvements including semantic HTML, ARIA landmarks, and proper roles
- Page-specific stylesheets for better organization and maintainability
- Simple, intuitive navigation between the two pages
- Clean, semantic markup and well-structured CSS

## How to Use

Open `index.html` or `donation.html` in any modern web browser to view the site locally.

## Development Notes

- Home page styles: `css/main.css`
- Donation page styles: `css/donation.css`
- Images and other assets are located in the `assets/` folder
- Media queries ensure seamless adaptation across mobile, tablet, and desktop screens
- Emphasis on semantic HTML5 and accessibility best practices
